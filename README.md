# Serverless Url Shortener Application built with AWS CDK

This is a Serverless application that intends to shorten your long url.

## Tools/Services
- AWS CDK
- AWS Lambda
- Amazon API Gateway
- Amazon Cognito
- Amazon DynamoDB
- AWS Serverless Application Model (SAM)
